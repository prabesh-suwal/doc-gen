Template Syntax

Templates use curly braces syntax similar to Carbone.io. Data is accessed with the d. prefix.

Feature Syntax Example
Variable {d.fieldName} {d.name}
Nested Property {d.object.property} {d.user.address.city}
Loop {#d.array}...{/d.array} {#d.items}{d.name}{/d.items}
Condition {#d.flag}...{/d.flag} {#d.isActive}Active{/d.isActive}
Formatter {d.value:formatter()} {d.date:formatDate('YYYY-MM-DD')}
Aggregation {d.array:agg('field')} {d.items:sum('price')}

Available Formatters

Date Formatters

  • formatDate(format) - Format date
  • addDays(n) - Add days
  • relativeTime() - Relative time

Number Formatters

  • formatNumber(decimals) - Decimal places
  • currency(symbol) - Currency format
  • percentage() - Percentage
  • thousands() - Thousands separator

Text Formatters

  • upperCase() - UPPERCASE
  • lowerCase() - lowercase
  • capitalize() - Capitalize
  • truncate(n) - Truncate text

Aggregations

  • sum(field) - Sum values
  • count() - Count items
  • avg(field) - Average
  • min(field) / max(field)

API Endpoints

POST /api/render One-time render with template upload
POST /api/render/:templateId Render from stored template
POST /api/templates Upload and save a template
GET /api/templates List all saved templates
GET /api/templates/:id Get template details
DELETE /api/templates/:id Delete a template
